# JOIN

| 连接类型    | 连接方式    | 说明                        |
| ------------ | ------------ | -------------------------- |
| INNER JOIN | 内连接     | 返回两个表中匹配的记录 |
| LEFT JOIN    | 左外连接 | 返回左表所有记录和右表匹配的记录   |
| RIGHT JOIN    | 右外连接 | 返回右表所有记录和左表匹配的记录  |






 - FULL OUTER JOIN / UNION    全外连接 返回两个表中所有记录，匹配不到的补 NULL

 - CROSS JOIN     笛卡尔积 返回两个表的笛卡尔积
